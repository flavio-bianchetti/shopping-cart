# Shopping Cart

Este repositório contém o projeto "_Shopping Cart_", requisito em atendimento ao módulo de Fundamentos - Javascript Assíncrono e Promises, da Trybe. O projeto consiste um carrinho de compras totalmente dinâmico, consumindo dados diretamente de uma API.

### Habilidades:
 - Fazer requisições a uma API (Application Programming Interface) do Mercado Livre;
 - Utilizar os seus conhecimentos sobre JavaScript, CSS e HTML;
 - Trabalhar com funções assíncronas;
